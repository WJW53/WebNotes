# 想到哪儿记哪儿

## 浏览器请求页面大致经历了哪些过程

1. 发送url，DNS查询，请求IP地址
2. TCP三次握手
3. 服务器响应内容(js html css img)
4. js时间线 : 按照成哥讲的js时间线理解js解析执行的过程就好
5. 渲染页面： DOM树 CSSDOM树，这俩合起来生成RENDER树，再布局，渲染
6. TCP四次挥手


## 浏览器的基本组成
