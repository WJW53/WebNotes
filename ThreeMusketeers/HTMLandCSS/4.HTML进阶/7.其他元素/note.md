# 其他元素

1. abbr

表示缩写词

2. time

提供给浏览器或搜索引擎阅读的时间

3. b (bold)

很古老的元素，以前是一个无语义元素，主要用于加粗字体。

4. q

一小段引用文本

5. blockquote

大段引用的文本

6. br(用的少，就不推荐使用)

无语义，主要用于在文本中换行

7. hr(用的更少)

无语义，主要用于分隔

8. meta

还可以用于搜索引擎优化(SEO)

9. link

通常用于链接外部资源(CSS、图标)

- rel属性：relation,链接的资源和当前网站的关系
- type属性：链接的资源的MIME(标准格式的字符串)类型
